# Markdown 实时编辑器

这是一个现代化、功能强大的 Markdown 实时编辑器，提供专业的编辑体验和丰富的功能特性。

## ✨ 核心功能

### 📝 编辑功能
- **实时预览**：左侧编辑，右侧实时渲染显示效果
- **智能同步滚动**：编辑器和预览区双向同步滚动，精确定位内容
- **行号显示**：编辑器左侧显示行号，便于定位和调试
- **自动保存**：内容自动保存到浏览器本地存储，永不丢失
- **统计信息**：实时显示字数、行数、光标位置等编辑信息

### 🎨 界面功能
- **多主题支持**：浅色、深色、蓝色、赛博朋克四种主题
- **响应式设计**：完美适配桌面和移动设备
- **面板调整**：可拖拽调整编辑器和预览区的宽度比例
- **面板控制**：可独立隐藏/显示编辑器或预览区

### 📤 文件操作
- **导入功能**：支持导入本地 Markdown 文件
- **多格式导出**：支持导出为 .md 和 .pdf 文件
- **Markdown 指南**：内置完整的 Markdown 语法参考

### ⚡ 高级特性
- **双向智能同步**：基于内容映射的精确滚动同步
- **快捷键支持**：提高编辑效率的键盘快捷键
- **自定义设置**：字体大小、字体样式、背景图片等个性化配置

## 🚀 快速开始

### 基础使用
1. **编辑内容**：在左侧编辑区输入 Markdown 文本
2. **实时预览**：右侧自动显示渲染后的效果
3. **智能同步**：开启同步滚动，编辑器和预览区实时同步定位

### 文件操作
4. **保存内容**：点击保存按钮或使用 `Ctrl/Cmd + S` 快捷键
5. **导入文件**：点击导入按钮或使用 `Ctrl/Cmd + O` 选择本地 Markdown 文件
6. **导出文档**：支持导出为 Markdown (`.md`) 和 PDF (`.pdf`) 格式
7. **学习语法**：点击问号按钮查看完整的 Markdown 语法指南

### 界面定制
8. **主题切换**：使用主题选择器或 `Ctrl/Cmd + D` 切换界面主题
9. **面板调整**：拖拽分割线调整编辑器和预览区宽度
10. **面板控制**：使用快捷键隐藏/显示编辑器 (`Ctrl/Cmd + /`) 或预览区 (`Ctrl/Cmd + .`)

### 高级功能
11. **同步滚动**：点击同步滚动按钮或使用 `Ctrl/Cmd + R` 开启双向同步
12. **快捷操作**：使用键盘快捷键提高编辑效率
13. **个性化设置**：在设置面板中自定义字体、背景等

## ⌨️ 快捷键参考

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + S` | 保存到本地存储 |
| `Ctrl/Cmd + O` | 导入 Markdown 文件 |
| `Ctrl/Cmd + E` | 导出为 Markdown 文件 |
| `Ctrl/Cmd + D` | 切换界面主题 |
| `Ctrl/Cmd + R` | 开启/关闭同步滚动 |
| `Ctrl/Cmd + /` | 显示/隐藏编辑器 |
| `Ctrl/Cmd + .` | 显示/隐藏预览区 |
| `Ctrl/Cmd + ,` | 打开设置面板 |
| `Ctrl/Cmd + ←/→` | 调整面板大小 |
| `Ctrl/Cmd + Shift + ←/→` | 快速调整面板大小 |

## 🛠️ 技术架构

### 核心技术栈
- **前端框架**：纯 HTML5 + CSS3 + Vanilla JavaScript
- **Markdown 解析**：[marked.js](https://marked.js.org/)
- **PDF 导出**：[html2pdf.js](https://html2pdf.hertzen.com/)
- **字体系统**：Google Fonts (Inter + JetBrains Mono)

### 核心算法
- **智能映射算法**：基于正则表达式的行号到元素映射
- **滚动同步算法**：双向位置计算和即时同步
- **防抖优化**：避免频繁操作和性能问题
- **状态管理**：防止循环触发和冲突处理

### 设计特色
- **CSS 变量系统**：完整的设计令牌体系
- **模块化架构**：组件化的代码组织
- **响应式布局**：Flexbox 和 Grid 布局系统
- **无障碍支持**：键盘导航和屏幕阅读器兼容
